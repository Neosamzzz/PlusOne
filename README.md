# 📱 PlusOne

一个极简的计数器 App，用于记录你每完成一次小任务或行为时的次数。

---

## ✨ 功能介绍

- 🧮 添加自定义计数项  
- ➕ 每次点击「+」按钮，数字就加一  
- ➖ 支持减少计数  
- 📝 左滑重命名或删除计数项  
- 🧼 清爽简洁、没有花里胡哨

---

## 🛠 技术栈

- Swift  
- SwiftUI  
- Xcode  

---

## 💡 设计理念

> 我们总是在做一些日复一日的小事，  
> PlusOne 想陪你默默记录每一次坚持的瞬间。

---

## 📦 使用说明

1. 打开 App  
2. 添加一个你想要记录的事件（比如：喝水、做俯卧撑、背单词）  
3. 每完成一次，就点一次「+」  
4. 看着数字慢慢增长，你会越来越有动力！
